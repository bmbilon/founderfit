## Deployment

This is a single-file HTML application with no backend dependencies. Simply serve `index.html` from any web server.

### Vercel Deployment

1. Connect this repository to Vercel
2. Vercel will automatically detect and deploy the static site
3. Configure your custom domain in Vercel project settings

### Local Development

Open `index.html` in any modern web browser. The app works completely offline.
